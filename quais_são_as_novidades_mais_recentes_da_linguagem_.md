# Tópico: Python 3.13
**TIPO:** Linguagem de Programação
**TAGS:** Python, Desenvolvimento, Tecnologia, Programação
**DATA_REF:** 2024/10

## Núcleo do Conhecimento
* **Lançamento do Python 3.13**
    * **Data:** 7 de outubro de 2024
    * **Recursos:** Interpretador interativo melhorado, suporte experimental para execução em modo de threads livres, compilador Just-In-Time
* **Melhorias no Interpretador**
    * **Interpretador Interativo:** Melhorias na interface do usuário e suporte a cores
    * **Mensagens de Erro:** Melhorias nas mensagens de erro para ajudar na depuração
* **Biblioteca Padrão**
    * **Novas Funções:** `base64.z85encode()` e `base64.z85decode()` para codificação e decodificação de dados Z85
    * **Melhorias:** No módulo `copy` com a função `copy.replace()` e no módulo `os` com funções para trabalhar com descritores de arquivos de notificação temporizada do Linux
* **Desenvolvimento**
    * **Suporte a Threads Livres:** Permite a execução de threads em paralelo para tarefas CPU-bound
    * **Compilador Just-In-Time:** Melhoria no desempenho da execução do código Python

## Recursos Experimentais
* **Modo de Threads Livres:** Suporte experimental para execução em modo de threads livres
* **Compilador Just-In-Time:** Compilador experimental para melhorar o desempenho da execução do código Python

* **Desenvolvedores:** Podem aproveitar os novos recursos e melhorias para criar aplicativos mais eficientes e escaláveis
